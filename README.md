## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express

## How to Run Locally
1. Install Node.js
2. Open terminal
3. Go to server folder  
   `cd server`
4. Install dependencies  
   `npm install`
5. Run server  
   `node server.js`
6. Open browser  
   `http://localhost:3000`
